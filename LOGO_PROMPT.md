# Aegisguard Logo — Gemini Image Generation Prompt

Copy-paste any of these prompts into Gemini (gemini.google.com → "Generate image"). Start with Prompt 1 (primary). Try variations if needed.

---

## Prompt 1 — Primary (Shield + Circuit)

```
Design a modern, professional logo for "Aegisguard" — an open-source AI agent security runtime. The logo should convey deterministic protection, trust, and technical sophistication.

Visual concept: A geometric shield shape (inspired by the Greek aegis — divine protection) with clean, angular lines. Inside the shield, embed a minimal circuit-board pattern or neural network node graph that transitions into a shield perimeter — symbolizing that security is woven into the runtime itself, not bolted on. The circuit lines should feel precise and deterministic, not organic or fuzzy.

Color palette: Deep navy blue (#1a2332) as the primary shield color, with electric cyan/teal (#00d4aa) accent lines for the circuit traces. Optional subtle gradient from navy to a slightly lighter blue at the top of the shield. White (#ffffff) can be used for contrast elements.

Style requirements:
- Flat/semi-flat design (no heavy 3D effects, no photorealism)
- Clean vector aesthetic suitable for scaling from 16px favicon to 1024px
- No text in the logo mark itself (the wordmark "aegisguard" will be set separately in a monospace or geometric sans-serif font)
- Minimal detail — should be instantly recognizable at small sizes
- Professional and serious tone — this is a security product, not a playful app
- Symmetrical or near-symmetrical composition
- Should work on both dark and light backgrounds

What to avoid:
- No locks, padlocks, or keyhole imagery (overused in security)
- No literal shields with crosses or medieval heraldry
- No robot/AI face imagery
- No gradients that won't reproduce in single-color printing
- No excessive detail that becomes mud at 32px

The overall feeling should be: "precise, trustworthy, engineering-grade protection." Think Cloudflare or HashiCorp level of design maturity, not a startup mascot.
```

## Prompt 2 — Variation (Aegis + Greek Mythology Minimal)

```
Design a minimalist logo for "Aegisguard," an AI security runtime library. Inspired by the Greek aegis (Zeus's divine shield that made its bearer invulnerable).

Visual concept: An abstract, geometric representation of a shield seen from the front, with a single stylized eye or focal point at its center (referencing Athena's aegis which bore the Gorgon's gaze — the idea that threats are stopped the moment they're seen). The eye should be extremely abstract — just a diamond or lens shape with a dot — not realistic. The shield outline uses sharp, angular facets like a cut gemstone, suggesting mathematical precision.

Colors: Monochromatic — works in pure black on white. If color is added: dark slate (#2d3748) with a single accent in electric green (#10b981) for the central focal point.

Style: Geometric, minimal, vector-clean. No more than 3-4 shapes total. Should be drawable with straight lines and simple arcs only. Suitable as a GitHub organization avatar, PyPI project icon, and documentation header.

Avoid: Swords, warriors, Greek columns, olive wreaths, literal eyes, photorealism, complex gradients, text in the mark.
```

## Prompt 3 — Variation (Abstract Lattice)

```
Design a logo for "Aegisguard" — a deterministic security runtime for AI agents. The core concept is a capability lattice where permissions can only narrow, never expand.

Visual concept: An abstract downward-pointing lattice or diamond grid structure that visually narrows from top to bottom — representing the attenuation principle (capabilities only shrink). The top is wider (more permissions), the bottom is a sharp point (fully constrained). The lattice nodes are connected by clean lines, and the overall silhouette forms a shield or chevron shape naturally.

Colors: Primary deep blue (#1e3a5f), accent teal (#0ea5e9), nodes in white or light cyan. Dark background version: the lattice glows subtly in teal on a near-black (#0f172a) background.

Style: Geometric, technical, precise. Feels like a data visualization or graph theory diagram that happens to look like a shield. Clean enough for a favicon, detailed enough for a hero image.

Avoid: Literal shields, locks, binary/matrix rain effects, excessive glow, photorealism.
```

## Prompt 4 — Wordmark + Icon Combined

```
Design a combined logo (icon + wordmark) for "Aegisguard," an open-source Python security library for AI agents.

Icon: A small geometric shield formed by two overlapping angular brackets (like code syntax < >) that create a shield silhouette where they intersect. This represents "security through code" — the shield IS the code. Clean, 2-3 colors max.

Wordmark: "aegisguard" in lowercase, set in a geometric sans-serif or monospace font (similar to JetBrains Mono, IBM Plex Mono, or Inter). The word "aegis" in the accent color (teal/cyan #06b6d4), "guard" in neutral dark (#1e293b). Letter-spacing slightly expanded for readability.

Layout: Icon to the left, wordmark to the right, vertically centered. Also works as icon-only (for favicons, avatars) or wordmark-only (for headers, documentation).

Colors: Dark navy (#0f172a), teal accent (#06b6d4), white for contrast.

Style: Developer-tool aesthetic. Clean, technical, no decoration. Should feel at home next to logos like Rust, Go, FastAPI, or Pydantic.
```

## Tips for Best Results

1. **Generate multiple variations** — run the prompt 3-4 times and pick the best
2. **Iterate** — take your favorite result and say "make the shield more angular" or "simplify the circuit pattern"
3. **Request SVG** — after finding a design you like, ask "recreate this as a clean SVG with flat colors"
4. **Test at sizes** — ask Gemini to show the logo at 512px, 128px, 32px, and 16px to verify readability
5. **Dark/light** — ask for both dark-background and light-background versions
